# Create AWS Instance

- AWS에서 나만의 가상화 환경을 만들기 위해서는 인스턴스를 생성해야한다.
- 인스턴스에는 OS, 사양, 보안 등의 설정과 함께 가상환경을 생성한다.
### **1. 먼저 AWS에 계정을 생성해야 한다.**
- 참고로 대학생에게는 등록된 학교에 대해서 일부 크레딧을 제공해준다.
- 계정이 있다고 생각하고 인스턴스 생성 방법을 기술한다.
